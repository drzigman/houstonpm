12.1 Explanation

Here's a quick explanation of some of the parts of this test script. 

    
   use Test::More tests => 6;

Load the Test::More module and tell it we expect to run 6 tests. 

    
   use List::Util qw/max/;

Load the module to test. 

    
   use strict;
   use warnings;

We know about these.
