16 What Was That?

Let's break that down: 

    
   __PACKAGE__->run( @ARGV ) unless caller();
    The __PACKAGE__ token is the name of the package where the module resides. This is important if you declare a package.
    Call the run subroutine with the command line arguments as a parameter.
    ... unless
    The caller() subroutine returns a true value.
      caller returns the package, filename, and line it was called from.
      At the level we called it, that means what file called us.
      If we are loaded as a module, we get reasonable values.
      If we are called as a script, no one called us, so false is returned.
