3.1 Complex Data Structures?

    A scalar can only contain a string, a number, or a reference.
    An array can only contain a list of scalars.
    A hash maps one set of scalars (the keys) to another set of scalars (the values).
    Before references, all arrays and hashes were one dimensional.
