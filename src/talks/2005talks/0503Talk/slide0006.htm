3.2 Complex Data Structures

    We can combine references, arrays, and hashes to make complex data structures.
    For example, we can use a hash to represent structured data. 
    
  my $person = {
      lastname => "Munster",
      firstname => "Herman",
      address => "1313 Mockingbird Lane",
  };
    By using references with arrays, we can have a list of people: 
    
  my @people = (
      {
          lastname => "Munster",
          firstname => "Herman",
          address => "1313 Mockingbird Lane",
      },
      {
          lastname => "Garibaldi",
          firstname => "Michael",
          address => "Babylon 5",
      },
      {
          lastname => "Carter",
          firstname => "Sam",
          address => "Cheyenne Mountain",
      },
  );
