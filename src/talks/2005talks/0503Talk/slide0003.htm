2.1 References to Anonymous Somethings

    You can also create references to things that don't have names. 
    
   # Anonymous array reference

   my $aref = [ 0 .. 5 ];

   # Anonymous hash reference

   my $href = { a => 1, b => 2, c => 5 };

   # Anonymous code reference

   my $cref = sub { print "hello, world\n"; };
