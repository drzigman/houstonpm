3 XS and SWIG

    Perl is written in C
    XS is the traditional method of writing C code to talk to Perl.
    SWIG is another framework for connecting C to scripting languages.
      Supports C and C++ on one side
      On the other side, supports: Perl, PHP, Python, Tcl, Ruby, C#, Common Lisp, Java, Modula-3 and OCAML
    Both involve a fair amount of set up and programmer overhead.
    We won't discuss them further.
