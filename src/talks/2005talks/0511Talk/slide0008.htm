5 Inlining Code

There are three ways to pass code to the Inline modules:

    Pass the code as a string 
    
    use Inline C => "C code here";
    Pass from the DATA file handle. 
    
    use Inline CPP => 'DATA';
    ...

    __DATA__
    __CPP__
    C++ code goes here
    New approach with source filtering. (Alpha software, be careful). 
    
   use Inline::Files;
   use Inline C => 'BELOW';
   ...

   __C__
   C code here.
