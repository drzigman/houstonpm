4.3 Installation Issues

    Inline::C - Must be the same compiler used to compile Perl.
    Inline::CPP - Comiler must be compatible with the C compiler used to build Perl.
    Inline::Java - The path to the j2sdk must be supplied to Makefile.PL as an argument. 
    
    perl Makefile.PL J2SDK=/path/to/your/j2sdk/installation
    Inline::Python - Need Python development files, not just Python runtime.
    Inline::Ruby - I never got this to work and don't have any idea why not.
    Inline::PUGS - Requires GHC (http://haskell.org/ghc/)
