6.3 Regexes and List Context

Matched text returned in list context. 

    
   my ($name, $value) = m/\s*(\w+)\s*=\s*(.*)/;

The value of $1 is assigned to $name and the value of $2 is assigned to $value. If there were a value for $3, it would have been discarded in this case.
