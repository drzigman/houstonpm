3.2 Currying (addendum)

The objects returned by indexing into $RE can be curried to generate more specific objects by adding modifying keys. For example, 

    
   my $intre = $RE{num}{int};

   $intre->{-base => 16}  # finds hexidecimal numbers
   $intre->{-base => 2}   # finds binary numbers
   $intre->{-places => 8} # finds eight-digit integers
