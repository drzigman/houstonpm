7.4 The New Constructor

This constructor is a bit more complicated. The initial lines are useful if this method is called as a method of an object. I also manipulate the class data. 

    
sub  new
{
    my $self = shift;
    my $class = ref $self || $self;

    ++$objCount;

    bless ClassSimple::new(), $class;
}

The two-parameter version of bless reblesses the object reference into the package specified by $class.
