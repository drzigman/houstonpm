4.3 More Unexpected Matches

    Null matches - .* matches everything...and nothing. 
    
    # Wrong way to convert multiple spaces into single spaces.
    $string =~ s/\s*/ /g;
    \s matches newline, even though . doesn't; 
    
    # Wrong to remove trailing spaces
    while(<>)
    {
        s/\s+$//;
        print;
    }
