4.1 Unexpected Matches

    Greedy matching
    Embedded words 
    
    print "found\n" if "Simplification" =~ /cat/;
    Accidental metacharacter use. 
    
    print "found\n" if "123.456" =~ /\d+.\d+/; # Expected
    print "found\n" if "123B456" =~ /\d+.\d+/; # Not
