2.5 Case translation

    Perl defines a set of escape sequences that change the case of following characters in a string.
    \U - uppercase all characters to the end of string or the next \E
    \u - uppercase the next character
    \L - lowercase all characters to the end of string or the next \E
    \l - lowercase the next character
    \E - turn off the previous \L, \U, or \Q
