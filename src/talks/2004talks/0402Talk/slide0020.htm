Extracting unique items 

    
    my @list = qw/foo bar baz foo quux bar foo/;
    print "@list\n";
    my %seen;
    foreach my $item (@list) {
        $seen{$item}++;
    }

Now, we could print out a list of the unique elements along with their frequency in the original list: 

    
    foreach my $key (keys %seen) {
        print "$key: $seen{$key}\n";
    }

Or, perhaps we just want a new array of the unique elements: 

    
    my @unique = keys %seen;
    print "@unique\n";
