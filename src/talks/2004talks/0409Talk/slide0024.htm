7.7.1 Extracting the Data

Extracting the regular expression and options is easy, we told the GUI which variables to fill with the input. So we just need to use those variables. 

    
  $f1->Entry( -textvariable => $regex, -width => 50 )

In this case the regular expression is a string in the variable $regex.

The Text block is a little harder. 

    
  my $text = $textWidget->get( '1.0', 'end' );

The get method uses indexes which are in the form line.char. There are also special indexes like end that points to the end of the text.
