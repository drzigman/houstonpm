Example 2

We can use an array of strings to initialize a hash with values of 1 for each key: 

    
    my @strings = qw/foo bar quux blah/;
    my %hash = map { $_ => 1 } @strings;
    foreach my $key (keys %hash) {
        print "$key:$hash{$key}\n";
    }
    __END__
    prints:
        foo:1
        blah:1
        quux:1
        bar:1
