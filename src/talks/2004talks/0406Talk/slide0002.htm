next last and redo

These can be very useful in conjunction with the loop control statements: next, last, and redo. Consider a situation in which you want to process lines of a file but you want to skip blank lines (or lines with only whitespace) and stop processing when you reach a line starting with 'STOP'. With traditional if statements may you do a variation of this: 

    
    while(<FILE>){
        if (/^STOP/){
            last;
        }
        if (/^\s*$/){
            next;
        }
        # ...do processing here
    }
