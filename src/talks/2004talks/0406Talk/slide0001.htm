Statement Modifiers and Loop Control

All of the flow of control constructs (loop and if statements) can be applied to single statements without using a block. For example: 

    
    print "$foo is less than $bar\n" if $foo < $bar;

    print "$_\n" for 1 .. 10;

    my @list = (1,2,3);
    my $sum = 0;
    $sum += $_ for @list;
    print "$sum\n";
