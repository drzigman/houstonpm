5.2 Configuration from Command Line

Use the -p option to choose a port and the -d option to set a document root directory. 

    
    my %opts = ();
    getopts( "p:d:", %opts ) or die "Invalid argumentn";

    my $port    = $opts{p} || 8080;
    my $docroot = $opts{d} || ".";

If no options are passed, we can default to reasonable values.
