8.2 Changed Request Processing

I've added statistics gathering code in various places to make server_stats work. The code that serves static pages has been factored into a new routine. 

    
	    $Stats{requests}++;

            my $path = $r->url->path();
	    if(exists $DynaPage{$path})
	    {
	    	$DynaPage{$path}->( $conn, $r, $path );
	    }
	    else
	    {
	    	handle_static_page( $conn, $r, $path );
            }

The exists test checks to see if we've defined this path as a dynamic one. If it is, we execute the routine associated with the path.
