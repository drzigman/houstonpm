15 Binding Parameters (Input)

DBI supports binding parameters which allows you to reuse statements with minor changes each time. 

    
  my $sth = $dbh->prepare( "SELECT * FROM People where id = ?" );

  $sth->execute( 1 );
    .
    .
    .
  $sth->execute( 2 );
    .
    .
    .

Each time we call execute, we can pass it a new set of parameters to replace the ?s in the SQL statement.
