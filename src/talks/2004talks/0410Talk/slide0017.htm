17 Simplifying non-SELECT Statements

You can use the do() method instead of prepare() and execute() if you do not need a statement handle. 

    
  $dbh->do( "DELETE from People" );

The only thing we need to know about this call is whether it succeeded, so we do not need a statement handle and can use the do() method.
