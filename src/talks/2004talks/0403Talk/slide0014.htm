Using a Hash.

If we want to store the options in a hash rather than separate variables we pass a reference to the hash as the *first* argument (different from the getopt() and getopts() functions). 

    
    my %opts;
    GetOptions(\%opts, 'a', 'help');

In this case, the options would be stored in $opts{a} and $opts{help}.
