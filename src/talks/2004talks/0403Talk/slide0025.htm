Simple program

The whole program is rather trivial, you specify a pattern and some files and it will print out each line containing the pattern (if the -not option is also specified, it will print out each line that does not contain the pattern).

The Getopt::Declare module also gives a few things for free. It predefines a usage statement that will be printed if the user specifies a -help (or even just -h) option, and it will print the version number of your program (assuming you defined a $::VERSION variable as we did above) if the program is invoked with a -version (or -v) option.
