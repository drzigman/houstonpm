Getopt::Std Module Functions.

The Getopt::Std module provides us with two functions: getopt() and getopts(). The getopt() function allows you to define single character options that will take arguments. You pass the function a string of the option characters and the function takes care of setting global variables of the form: $opt_f or $opt_d (where f and d were characters passed to the function). To use this under 'use strict' you want to declare your global option variables prior to calling getopt(): 

    
    #!/usr/bin/perl -w
    use strict;
    use Getopt::Std;
    use vars qw/$opt_f $opt_d/;
    getopts('fd');

    print "-f is $opt_f\n" if defined $opt_f;
    print "-d is $opt_d\n" if defined $opt_d;
    print "other args: @ARGV\n";
    __END__

You can call the above as follows (assuming the program is named 'opt.pl'): 

    
    perl opts.pl -f somefile -d12 other
    -f is somefile
    -d is 12
    other args: other
