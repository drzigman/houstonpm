Parsing Command Line Options - Getopt::Long

The Getopt::Long module provides for parsing long option names and bundling of option switches, as well as for providing a rudimentary type check on option arguments (it provides more configurability, but this is all we'll cover here).

The function provided by this module is GetOptions(), and there are a few differences in how this function is called compared to the functions we discussed before.
