Using Hashes

To avoid setting global variables, you can also specify a reference to a hash as a second argument to the getopt() function (and the getopts() function which we'll look at next): 

    
    #!/usr/bin/perl -w
    use strict;
    use Getopt::Std;
    my %opts;
    getopt('fd',\%opts);

    print "-f is $opts{f}\n" if defined $opts{f};
    print "-d is $opts{d}\n" if defined $opts{d};
    print "other args: @ARGV\n";
    __END__
