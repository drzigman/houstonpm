5.3 Start the Server

Launch the server connected to the supplied TCP/IP port. Let the user know the server is launched and how it is configured. 

    
    my $daemon = HTTP::Daemon->new( LocalPort => $port ) or die;
    print "Please contact me at: <URL:", $daemon->url(), ">n",
          "Documents in '$docroot'n";
