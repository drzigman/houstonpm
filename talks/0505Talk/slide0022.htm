12.1 Changing Delimiters

    Change delimiters on m// and s/// to simplify expressions. 
    
   # Which is better
   $file =~ s/^.*\/([^\/]*)$/$1;
      #or
   $file =~ s#^.*/([^/]*)#
    Using ' for delimiters stops variable interpolation.
    Don't forget matching delimiters (), {}, etc.
