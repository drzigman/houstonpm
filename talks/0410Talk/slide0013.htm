13 Handles

There are two kinds of handles that you need to become familiar with in the DBI.

  1. Database handle ($dbh)
  2. Statement handle ($sth)

The database handle is your interface to the database connection and any global configuration.

A statement handle is associated with each SQL statement. These can be used to control the statement, and get return information, metadata, etc. associated with a particular statement.
