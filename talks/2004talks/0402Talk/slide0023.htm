Data Input Specification

Consider a problem where you want to read a data file and print out a report based on certain fields in the data. Let's assume the file is an inventory parts list with data of the form: 

    
    part_number:price:category:supplier

And, further, if the supplier name has an asterisk before it that means the item isn't kept in stock but needs to ordered directly. A sample of this file looks like: 

    
    123:17.00:widgets:ACME
    214:15.50:gadgets:*ACE
    313:2.95:thingamabobs:TMG
    222:4.95:widgets:*A-DS
    524:0.15:gadgets:ACE
