Benchmark.pm

The other main method of benchmarking is to use the Benchmark.pm module included with your perl distribution. There are a few ways to use this module but we'll only consider one common way at this time. 

    
    #!/usr/bin/perl -w
    use strict;
    use Benchmark;

    my $count = 100000;
    timethese($count, {
        cntrl => sub {$_ = 'foobar'},
        subst => sub {$_ = 'foobar'; s/[aeiou]//g},
        trans => sub {$_ = 'foobar'; tr/aeiou//d},
        });
    __END__
