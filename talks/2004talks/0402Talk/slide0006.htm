Benchmark summary.

The Benchmark module has several other ways of timing routines or strings of code -- notably, if you import the 'cmpthese' function and use it instead of the 'timethese' function, you'll get the same results along with a comparison table showing the percent time differences among the routines tested. As always, 'perldoc Benchmark' will give you the full documentation for the module.
