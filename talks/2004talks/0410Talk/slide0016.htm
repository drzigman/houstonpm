16 Binding Columns (Output)

Although it's easy to fetch arrays of output as shown above, sometimes you know right where you want the data to be put. Binding columns helps with that. 

    
  $sth->bind_col( 1, $id );
  $sth->bind_col( 2, $surname );
  $sth->bind_col( 3, $given );
  $sth->bind_col( 4, $position );

  while($sth->fetch)
  {
      # Do something with $id, $surname, $given, and $position
  }

Note that column numbers start with 1.
