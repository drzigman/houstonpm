lib

This is the module to use if you are forced to use private libraries because your Sys Admin won't install needed modules.

This is a small simple module which simplifies the manipulation of @INC at compile time.

It's typically used to add extra directories to perl's search path so that later "use" or "require" statements will find modules which are not located on perl's default search path.

Adding directories to @INC

The parameters to "use lib" are added to the start of the perl search path. Saying 

    
    use lib LIST;

is almost the same as saying 

    
    BEGIN { unshift(@INC, LIST) }

For each directory in LIST (called $dir here) the lib module also checks to see if a directory called $dir/$archname/auto exists. If so the $dir/$archname directory is assumed to be a corresponding architec- ture specific directory and is added to @INC in front of $dir.
