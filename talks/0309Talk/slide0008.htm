Grabbing and Changing Substrings within a String

We had a little discussion about the last substring example. The original question was is there any difference between these two lines? 

    
 substr($string,3,3,'quux');
 substr($string,3,3) = 'quux';

It seemed to me that the values of the two expressions would be different, but I wasn't certain. We know the value of the first expression is 'bar'. I suspected that the value of the second expression would be 'quux'. However, I was amazed to find that the value is the substr() of the new string. In this case, the value is 'quu'. 

    
  $string = 'foobarbaz';
  $data   = substr($string,3,3,'quux');
  print "$string:$data\n"; # prints: fooquuxbaz:bar

  $string = 'foobarbaz';
  $data   = substr($string,3,3,) = 'quux';
  print "$string:$data\n"; # prints: fooquuxbaz:quu
