Using the tr operator to count characters.

Sometimes you want to count particular characters in a string. The tr/// operator replaces characters in a given list with corresponding characters in a second list. 

    
    tr/list/replacementlist/.

If no replacement list is given, this operator automatically copies the first list to use as the replacement list (meaning every listed character gets replaced by itself). The tr/// operator returns the number of replaced characters so you can use it to count particular characters: 

    
    $string = 'The quick brown fox jumped over the lazy dog';
    $vowel_count = $string =~ tr/aeiou//;
    print $vowel_count; # prints: 12

The length() function is documented in the perlfunc pod-page see: perldoc -f length. For more on the tr/// operator see the perlop pod-page.
