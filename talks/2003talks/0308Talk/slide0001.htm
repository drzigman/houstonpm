Using Perl as a Command Line Tool

Perl has many command line switches that affect its behavior when you run it. The -e switch will accept a string of Perl code (it may be many statements long) and use it as the script to run: 

    
    perl -e 'print "hello world\n"'

Any additional arguments are passed to the given script in the @ARGV array: 

    
    perl -e 'print "You passed in: @ARGV\n"' foo bar baz

The -e switch is used to tell perl not to look for a filename in the argument list and that the program is contained in one line.
