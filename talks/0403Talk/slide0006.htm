Summary of Manual Options

We have only considered very simple cases thus far. What if we wanted to be able to have an optional space between the -n and the numerical argument? Or what if we wanted to allow for processing multiple options as a 'bundle'(where -nav would mean -n -a -v)?. Next we'll introduce the Getopt::Std module, which comes with the standard perl distribution and does all the hard work of processing simple options for us (including options with arguments, and option bundling).
