Predefined Formats

This module also allows you to specify a few predefined formats for option arguments to take: 

    
    -x <num:n>    the :n means the argument must be a number
    -x <num:i>    the :i means the argument must be an integer
    -x <num:+i>   must be a positive integer
    -x <str:s>    must be a string (default case)
    -x <file:if>  input file (must be readable file or '-' for STDIN)
    -x <file:of>  output file (must be writable or not exist or '-'
                  for STDOUT)

There are a few other builtin specifier types, and you can define new ones using the full power of Perl's regular expressions.
