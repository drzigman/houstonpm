Processing Sequence

In the program, we used a call to the Getopt::Declare::version() function directly to print a version number. The interesting thing to note is when this is printed. All of the option actions take place immediately when each option is encounted, except for deferred options, which take place as soon as the command line is parsed and before any further code is executed. Hence, in our example, all of the files are processed by the time we get to the statement which prints the version number.
