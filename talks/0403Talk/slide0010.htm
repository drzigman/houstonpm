Using Arguments.

The getotps() function allows you to define both options that act as switches and those that take arguments. In this case, in the string argument, each character that you want to take an argument is followed by a ':' character: 

    
    #!/usr/bin/perl -w
    use strict;
    use Getopt::Std;
    my %opts;
    getopts('f:d',\%opts);

    print "-f is $opts{f}\n" if defined $opts{f};
    print "-d is $opts{d}\n" if defined $opts{d};
    print "other args: @ARGV\n";
    __END__

In this case, the -f option will take an argument and the -d option acts a switch: 

    
    perl opts.pl -f f_arg -d  other
    -f is f_arg
    -d is 1
    other args: other

The getopts() version also gives you a warning if the program is called with options that haven't been defined (for some reason, the getopt() function doesn't give us such warnings).
