Changing the Autosplit Delimiter on the Command Line

When using the -a switch to autosplit into the @F array, you may also use the -F switch to specify a delimiter to split on (the default is to split on space). 

    
    perl -an -F/:/ -e 'print "$F[4]\n"' filename

This will split colon-separated fields rather than fields separated by whitespace.

Command line switches are documented in the perlrun pod-page.
