The -p switch. 

    
   perl -pe 's/Street/St./g' filename

This will print every line from the file, and occurrences of 'Street' will be replaced with 'St.'. The implicit loop means the above script is interpreted as: 

    
    while(<>) {
        s/Street/St./g;
        print or die "-p destination: $!\n";
    }
