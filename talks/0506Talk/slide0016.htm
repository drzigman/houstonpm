4.6 Still More Mismatches

    Nested structures 
    
   # Does not match a nested set of parentheses.
   /\((.*?)\)/;
    \10 matches the first captured text and a 0, unless there are ten or more sets of capturing parentheses, then it matches the tenth captured text.
