5.2 Anchors are your friends

    If the regex starts with ^, it must match at the beginning of the string (or beginning of line with the m modifier).
    If the regex ends with $, it must match the end of the string (or end of line with the m modifier).
    The other anchors \A, \z, \Z, and \G, work similarly.
    If the expression doesn't match there, no need to work the way through the rest of the string.
