19 Automating Testing Advantages

    Testing is boring
    Automated testing simplifies testing.
    Automated testing gives you more confidence that your code is working.
    The more tests you add, the more confident you will be.
    Running the test is so easy, you'll be likely to do it after every change.
    This helps you find bug close (in time) to when you make them.
