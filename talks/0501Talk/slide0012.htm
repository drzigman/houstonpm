5.3 Define Methods

What good is an object without methods? The object reference is passed to all methods as the first parameter. It is called $self by convention. 

    
sub  add
{
    my $self = shift;
    $self->{counter} += shift;
}

sub  get
{
    my $self = shift;
    $self->{counter};
}
