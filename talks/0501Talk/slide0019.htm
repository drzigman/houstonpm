7.5 More Methods

Create more methods, just like in the previous class. 

    
sub  set
{
    my $self = shift;
    $self->{counter} = shift;
}

sub  incr
{
    my $self = shift;
    $self->add( 1 );
}

Notice the use of a method from the base class.
