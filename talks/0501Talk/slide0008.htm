4.2 How to Use a Reference

You use references by dereferencing them. 

    
  print $$sref, ":", ${$sref}, "\n";

  print keys %$href, "\n";
  print keys %{$href}, "\n";
  print $href->{name}, ",", $$href{name}, ",", ${$href}{name}, "\n";

  print @$aref, "\n";
  print @{$aref}, "\n";
  print $$aref[0], ",", ${$aref}[0], ",", $aref->[0], "\n";

  &$cref();
  &{$cref}();
  $cref->();

I prefer the "->" form for access. Object methods use a similar syntax.
