    Programming with objects
    Programming using the concepts of OO
Abstraction
Ignoring details unless they are needed.
Encapsulation
Expose the what, not the how.
Polymorphism
Ability to treat different types as the same.
Inheritance
Definition of one object can extend or specialize the definition of another.
