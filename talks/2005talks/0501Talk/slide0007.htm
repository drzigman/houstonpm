4.1 How to Get a Reference

    The backslash operator
    Anonymous Data

Here are examples: 

    
  $sref = \$var;

  $href = \%hash;
  $href = { name => 'value' };

  $aref = \@array;
  $aref = [ 'fred', 'barney' ];

  $cref = \&func;
  $cref = sub { return 'foo'; };
