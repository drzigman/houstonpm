7.4 Here Documents

    A Here Document is a special syntax usually used for long, multi-line strings. 
    
  my $usage = <<"EOM";
  Usage: $0 [-i -l -s] {filename} ...

  where:  -i   process as integer
          -l   process as long
	  -s   process as string
  and   {filename} is the name(s) of one or more files to process.
  EOM
    The (possibly quoted) terminating string after the << determines where the here document ends.
    The terminating string must occur on a line by itself (without quotes or whitespace) to work.
