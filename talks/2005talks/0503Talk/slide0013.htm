5.2 Using the New Commands Hash

    To use the new features, we need a somewhat more complicated dereference. 
    
    if(exists $Commands{$word})
    {
	if(@Stack < $Commands{$word}->{count})
	{
	    print "Stack underflow!\n";
	    @Stack = ();
	    next LINE;
	}
	$Commands{$word}->{code}->();
    }
    After finding the hashref associated with the key, we dereference a second time to get the actual data.
