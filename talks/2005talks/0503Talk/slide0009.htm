4.2 Using the Commands Hash

    The key to the %Commands hash is how it is used. 
    
    if(exists $Commands{$word})
    {
	$Commands{$word}->();
    }
    We test to see if a word is in the hash with exists.
    If so, we execute the coderef associated with the string.
