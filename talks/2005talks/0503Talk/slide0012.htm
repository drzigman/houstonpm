5.1 The New Commands Hash

    We have extended the values in the %Commands hash to be references to hashes.
    These new extended structures store more information. 
    
    "+" =>
        {
           code => sub { push @Stack, pop( @Stack ) + pop( @Stack ) },
	   help => "adds the top two numbers on the stack",
	   count => 2,
	},
    Once again a command string is the key, but the value now holds
      a coderef,
      a help string,
      a required stack depth.
