2.5 The Regexp::Common Solution

Using Regexp::Common, the solution looks like this. 

    
  use Regexp::Common;

  /$RE{num}{int}{-sep => ','}{-group => 3}{-keep}/g;

This manages an exact match on positive and negative integers with commas grouping each set of three digits.
