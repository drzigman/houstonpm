4 Regex Basics

    Single non-meta-character matches itself
    Special characters: (\n, \r, \t, \07, \x20, \cA, etc.)
    Character class matches any single character in the class ([aeiou])
    Negated Character class matches any single character not in the class ([^aeiou])
    Alternatives: (dog|cat)
    Basic meta-characters: ., \w, \W, \s, \S, \d, \D.
    Quantifiers: *, +, ?, {n}, {n,}, {n,m}
    Clustering and capturing: (\d+)
    Backreferences: \1, \2, etc.
    Basic anchors: ^, $
    Simple options: i, g
