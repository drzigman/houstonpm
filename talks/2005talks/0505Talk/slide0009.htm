5.4 Lookahead

    (?=) - positive lookahead 
    
  /(\d+)(?=\s)/;  # matches a number only if it followed by whitespace
    (?!) - negative lookahead 
    
  /(\d+)(?!\w)/;  # matches a number that is not followed by a word character
