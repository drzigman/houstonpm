14 Test::Harness

Laziness is a Perl virtue. 

    
t/max....ok
t/min....ok
t/sum....ok
All tests successful.
Files=3, Tests=17,  0 wallclock secs ( 0.13 cusr +  0.02 csys =  0.15 CPU)

The Test::Harness module can run multiple test scripts and summarize the output. 

    
#!/usr/bin/perl

use Test::Harness;
use strict;
use warnings;

runtests( @ARGV );
