12.4 Bad Output

If there were errors the output from this script would look like this. 

    
1..6
ok 1 - empty list
not ok 2 - max item is last
#     Failed test (t/max.t at line 8)
#          got: '5'
#     expected: '2'
not ok 3 - max item is first
#     Failed test (t/max.t at line 9)
#          got: '5'
#     expected: '3'
not ok 4 - max item is middle
#     Failed test (t/max.t at line 10)
#          got: '5'
#     expected: '4'
ok 5 - all the same
ok 6 - only one
# Looks like you failed 3 tests of 6.
