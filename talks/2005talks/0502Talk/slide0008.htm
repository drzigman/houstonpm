8 Approach: Testing with Diff

Say I have some code that gives the correct answer, but needs to be cleaned up.

    Black box regression testing is a good approach in this case.
    1. Run the code to generate an output that we save to a file.
    2. Make a change.
    3. Generate a new output file.
    4. Compare the output to the reference file.
    5. If it matches, go back to 2.
    6. If it doesn't match, either back out the change or fix the error,
    7. Go to 3.
