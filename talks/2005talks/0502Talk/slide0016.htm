12.2 The Actual Tests

The ok() test succeeds if its first argument is true. 

    
   ok( !defined( max() ),   "empty list" );

The is() test succeeds if its first argument matches its second. 

    
   is( max( 1,2,3,4,5 ), 5, "max item is last" );

Test::More supports several other test methods:

    ok - boolean test
    is, isnt, is_deeply - compare two items with reporting
    like, unlike - test against a regular expression
    cmp_ok - arbitrary binary test
    can_ok, isa_ok - object or reference testing.
    use_ok, require_ok - module load testing
    pass, fail - for your use in special tests
