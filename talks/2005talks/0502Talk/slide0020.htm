13 How Does This Help?

    Simple framework for testing subroutines.
    Test is easy to run, so you might do it.
    Output is not too hard to figure out.
    Good for catching regressions: The test worked before this change, now a particular test fails.
    Makes finding bugs easier, because you get feedback quickly.
    But, it still requires running each test file individually.
    You also still have to scan the output looking for 'not ok' cases.
