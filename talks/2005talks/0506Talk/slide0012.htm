4.2 Greedy Matching

    Using a greedy quantifier is sometimes surprising: 
    
  $string = "This is a <em>string</em> with embedded <b>tags</b>.";

  # Remove html tags
  $string =~ s/<.*>//g;

  # $string is now "This is a."
    Non-greedy matching may match less: 
    
  $string = "This is a <em>string</em> with embedded <b>tags</b>.";

  # Remove html tags
  $string =~ s/<.*?>//g;

  # $string is now "This is a string with embedded tags."
