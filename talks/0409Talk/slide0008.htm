4.4 Add some widgets 

    
   $mw->Label( -text => 'Hello World' )->pack;
   $mw->Button( -text => "Done", -command => sub { exit } )->pack;

Next, we need to add some widgets. I've added a text label and a button. The button will shut down the program when it is clicked. We'll come back to that in a minute.

We execute the pack method on each widget to position it on its parent.
