7.7 Testing the Regexp 

    
  $mw->Button( -text => 'Test', -command => &test_regexp )->pack;

When the Test button is clicked:

  1. We extract the data from the GUI.
  2. Test the supplied regular expression.
  3. Update the GUI with the results.
