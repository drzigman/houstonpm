7.6 Building the Display

Building the display is pretty complicated so we'll look at build_window in pieces. In addition to new widgets, this code shows an important technique, the use of Frames. 

    
  my $mw = shift;
  my $f = $mw->Frame();

  # Regular Expression Entry
  my $f1 = $f->Frame();
  $f1->Label( -text => 'Regular Expression:' )->pack( -anchor => 'nw' );
  $f1->Entry( -textvariable => $regex, -width => 50 )
             ->pack( -side => 'left', -anchor => 'n' );
  $f1->pack( -side => 'left', -anchor => 'n' );

Here we've added an Entry widget for the regular expression.
