4.1 The Commands Hash

    Most of the code is straightforward.
    The command hash is different. 
    
  my %Commands =
  {
      "+" => sub { push @Stack, pop( @Stack ) + pop( @Stack ) },
      ...
    "quit" => sub { exit; },
    "help" => \&help,
  };
    This hash maps strings to an subroutines.
    Most of the coderefs are for anonymous routines.
    The last entry maps to a normal named subroutine.
