7.5 More Here Documents

    If the terminating string is single-quoted, so is the here document.
    If the terminating string has no quotes, the here document is processed as if double-quoted.
    A here document can be used anywhere a string can be used. 
    
  printf( <<'EOF', $x, $y, $size );
    The object in question is located at (%d,%d).
    It's size is %5d.
  EOF
