4.5 More Mismatches

    . doesn't match newline (unless s modifier is in effect). 
    
  $html =~ m/<[Pp]>(.*?)<\/[Pp]>/;
    Negated character classes must match a character. 
    
   # Wrong way to find the last vowel in a string.
   /.*([aeiou])[^aeiou]/;
    Doesn't match if last letter is a vowel.
