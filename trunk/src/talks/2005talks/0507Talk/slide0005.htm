2.3 First Wrinkle

As you look through comparing inputs and outputs, you discover that the numbers in your target text have commas. A quick-and-dirty solution would be: 

    
   /(-?[\d,]+)/g

This matches any cluster of digits and commas. Unfortunately, this is not a perfect fit because it also matches single commas.

Let's try to be more specific: 

    
   /(-?\d[\d,]+)/g

This will only match if there is at least one digit at the beginning. Unfortunately, it also matches a number followed by a comma.
