4 Functional Usage

There is also a supported subroutine-based interface. The subroutine names take the form RE_ followed by an underscore separated list of identifier keys. For example, 

    
   RE_num_int( -sep => ',', -group => 3 );

returns a regex that matches integers. As you can see the flags are passed as arguments to the subroutine.
