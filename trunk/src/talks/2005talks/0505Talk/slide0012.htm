6.2 After the Regex

The matched text may be used after a regex match with $1, etc. 

    
   # swap the first two words
   $string =~ s/(\w+) (\w+)/$2 $1/;

   # extract a (US) phone number
   if($string =~ /(\(?\d\d\d\)?[- ]?\d\d\d[- ]?\d\d\d\d)/)
   {
       $phone = $1;
   }

Using the more verbose if statement to clarify the example.
