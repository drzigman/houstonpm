12.5 Knowing What to Discard

    Split matches what we want to discard. 
    
   my @fields = split( /:/ );  # password file
   my @fields = split( /,/ );  # simple CSV
   my @fields = split( /\t/ ); # tab-delimited fields
   my @fields = split( " " );  # whitespace separated.
    Delimiters can be full-blown regexes.
