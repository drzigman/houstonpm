8.2 Diff Testing Example

A typical successful diff test session would run like this: 

    
   $ prog > test.out
   $ diff standard.out test.out
   $

A typical unsuccessful diff test session would run like this: 

    
   $ prog > test.out
   $ diff standard.out test.out

   2c2
   < Value: 1.0000
   ---
   > Value: 1.0001
