7 Perl Testing Frameworks

    Test::Simple - Basic framework for testing.
    Test::More - More advanced version of Test::Simple.
    Test::Harness - A driver for interpreting the output of other Test modules.
    Test::Unit - Object-Oriented testing framework based on JUnit.
    Devel::Cover - Coverage testing. Helps you to know if all of the code is tested.
