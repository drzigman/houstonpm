3 Kinds of Tests

Unit testing
Test a class, module, or function to verify functionality.
Component testing
Test an integrated aggregate of one or more units.
Integration testing
Connecting larger and larger groups of components together and testing how they work together.
System testing
Test the entire program or system for overall functionality.
Functional testing
Test the system for a defined set of functionality.
Performance testing
Test the speed or throughput of the system.
Regression testing
Test for reappearance of old bugs.
