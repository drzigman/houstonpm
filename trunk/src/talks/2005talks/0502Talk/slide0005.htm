5 Testing by Diff

First level of automation is to compare the output with a known good output.

    A good example of Black Box Testing
    Works best when it can be done by computer (e.g. diff).
    Requires a known-good output.
    Can be tripped up by incidental differences like current date and time.
    Can be run as part of an automated test suite or regression test.
