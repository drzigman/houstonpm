7.1 Basic Quoting

There are three basic quoting operators in Perl.

    Double quotes: "hello".
      Interpolates variables.
      Supports escape sequences (e.g. "\n", "\t")
    Single quotes: 'hello'
      Does not interpolate variables.
      Only supports two escape sequences ('\\' and '\'')
    Back ticks: `echo hello`
      Interpolates variables.
      Supports escape sequences (e.g. "\n", "\t")
      Passes resulting string to the shell for interpretation taking on the value of the output.
