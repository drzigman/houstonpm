5.3 Automatic Help Message

    The help message is now generated directly from the hash. 
    
   foreach my $cmd (sort keys %Commands)
   {
       printf "  %-5s %s\n", $cmd, $Commands{$cmd}->{help};
   }
    We walk the %Commands hash and display the messages we find.
    Note the double dereference, again.
