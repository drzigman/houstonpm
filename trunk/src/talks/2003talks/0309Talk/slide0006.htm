The splice function

The splice() function can remove or add elements to any arbitrary location in an array. The full syntax is: 

    
    splice ARRAY, OFFSET, LENGTH, LIST

The last two arguments are optional. This function removes LENGTH elements from the ARRAY starting at element OFFSET. If the LENGTH argument is not given, it removes everything from the OFFSET to the end of the array. If the LIST if given, then the removed elements are replaced by the given list. 

    
    my @array = (1,2,3,4,5);
    splice(@array, 1, 2);
    print "@array\n";    # prints: 1 4 5
    splice(@array, 1, 0, 42, 43, 44);
    print "@array\n";    # prints: 1 42 43 44 4 5

Notice how using a zero LENGTH parameter allows you to insert elements into the array without removing any existing elements.
