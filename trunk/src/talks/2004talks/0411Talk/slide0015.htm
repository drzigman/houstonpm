8.1 What's Different?

This hash maps the URL paths to subroutines. This will be needed for any dynamic webserver we build this way. 

    
    my %DynaPage =
       (
           "/stats" => &server_stats,
           "/hello" => &dynamic_hello,
       );

This hash stores information that needed by the server_stats subroutine. It is specific to this particular application. 

    
    my %Stats = ( start     => time(),
                  requests  => 0,
	          static    => 0,
	          notfound  => 0,
	          forbidden => 0,
                );
