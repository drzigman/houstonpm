5.4 Wait for connections...

Wait for a connection. Process it when it comes in. 

    
    while (my $conn = $daemon->accept())
    {
        eval
        {
          # ...
        };

        print STDERR "Caught exception: $@" if $@;

        $conn->close();
        undef($conn);
    }

The eval allows us to catch errors that would have been fatal, print a message, and continue.

We always want to close the connection when we're finished.
