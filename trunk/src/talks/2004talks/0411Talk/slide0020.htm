8.6 Second Dynamic Page

This page takes parameters from the request to update the page. 

    
    sub dynamic_hello
    {
        my $conn = shift;
        my $r    = shift;

        my $name = get_params( $r )->{name} || "request #$Stats{requests}";

        my $resp = new HTTP::Response();
        $resp->header( "Content-Type" => "text/html" );
        $resp->content( <<"EOS" );
    <html>
    <head>
      <title>Dynamic Hello World</title>
    </head>
    <body bgcolor="white">
    <h1>Hello, $name</h1>

    <p>This is dynamically generated text.</p>
    </body>
    </html>
    EOS

        $conn->send_response( $resp );
    }

Notice that the output is valid HTML as well.
