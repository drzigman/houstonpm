Main Difference.

One main difference is that you specify each option character (or name) as a separate argument (rather than a single string of option characters). To associate an option with a variable, you pass the option string followed by a reference to the variable. 

    
    #!/usr/bin/perl -w
    use strict;
    use Getopt::Long;
    my ($opt_a, $opt_help);
    GetOptions('a', \$opt_a, 'help', \$opt_help);
    print "-a is $opt_a\n" if defined $opt_a;
    print "--help is $opt_help\n" if defined $opt_help;
    __END__

We can now call this program (saved as testopt.pl) as follows: 

    
    perl testopt.pl -a          <-- a option is on
    perl testopt.pl --help      <-- help option is on
    perl testopt.pl --help -a   <-- both options are on
