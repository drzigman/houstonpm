Parsing Command Line Options - Getopt::Declare

The Getopt::Declare module is a very versatile, and relatively easy to use, command line parser module. It actually defines a grammer which you use to create a specification which it then uses to build a custom parser for your command line arguments. Sound complicated? It is, on the inside. But on the outside it's pretty straighforward, and we'll only be looking at the outside.
