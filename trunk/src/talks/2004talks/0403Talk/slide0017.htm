Bundling Options.

This only applies to single character options and must be enabled by making a call to the Getopt::Long::Configure() function as follows: 

    
    Getopt::Long::Configure('bundling');

Now, single character options may be bundled. Also, option values may be inserted into the bundle -- so if we did: 

    
    Getopt::Long::Configure('bundling');
    GetOptions(\%opts, 'h=i', 'w=i', 'a', 'b', 'c', 'help');

We could then call the program in any of the following ways: 

    
    testopt.pl -a -b -c -h 12 -w 5 --help
    testopt.pl -abc -h 12 -w 5 --help
    testopt.pl -abc -h12w5 --help
    testopt.pl -ah12w5bc --help
