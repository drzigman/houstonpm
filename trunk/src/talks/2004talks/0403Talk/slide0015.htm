Specifying Values For Options.

If we want options that take values we need to specify if the value is optional or mandatory, and if the value should be an integer, a floating point number, or a string. The following table shows the various specifications for an option named 'a': 

    
    option spec        meaning
        a              no value
       a:i             optional integer value
       a=i             mandatory integer value
       a:f             optional float value
       a=f             mandatory float value
       a:s             optional string value
       a=s             mandatory string value
