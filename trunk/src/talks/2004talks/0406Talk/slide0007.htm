The /x modifier to make it more readable 

    
    sub commify {
        my $input = shift;
        $input = reverse $input;
        $input =~ s{
                     (\d\d\d)    # match 3 digits into $1
                     (?=\d)      # if the are followed by a digit
                     (?!         # and they are not followed by:
                        \d*      # zero-or-more digits
                        \.       # followed by a decimal point
                     )
                   }{$1,}gx;
        return scalar reverse $input;
    }

All of the whitespace in the above regex is ignored as is anything following the comment symbol #. The above regex can be written on one line as: s{(\d\d\d)(?=\d)(?!\d*\.)}{$1,}g.
