CGI::Fast

If you can't get mod_perl installed in apache, this is the other way to have your programs run fast.

CGI::Fast is a subclass of the CGI object created by CGI.pm. It's specialized to work well with the FastCGI standard, which greatly speeds up CGI scripts by turning them into persistently running server processes. Scripts that perform time-consuming initialization processes, such as loading large modules or opening persistent database connections, will see large performance improvements.
