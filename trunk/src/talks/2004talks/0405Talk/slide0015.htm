File::Copy

The File::Copy module provides two basic functions, "copy" and "move", which are useful for getting the contents of a file from one place to another.

COPY

    The "copy" function takes two parameters: a file to copy from and a file to copy to.
    An optional third parameter can be used to specify the buffer size used for copying.
    You may use the syntax "use File::Copy "cp"" to get at the "cp" alias for this function. The syntax is exactly the same.

MOVE

    The "move" function also takes two parameters: the current name and the intended name of the file to be moved. If the destination already exists and is a directory, and the source isn't a directory, then the source file will be renamed into the directory specified by the destination.
    If possible, move() will simply rename the file. Otherwise, it copies the file to the new location and deletes the original.
