7.7.3 Handling the Captures

Before we attempt to match, we clear the captures Listbox. 

    
  $listWidget->delete( 0, 'end' );

After matching, we insert any captures into the Listbox. 

    
      for(my $i=1; $i < @-; ++$i)
       {
	$listWidget->insert( 'end', substr( $text, $-[$i], $+[$i]-$-[$i] ) );
       }
