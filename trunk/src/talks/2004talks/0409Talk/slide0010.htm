4.6 What was that sub thing on the Button? 

    
   $mw->Button( -text => "Done", -command => sub { exit } )->pack;

The -command parameter to the Button constructor takes a Perl code reference. This coderef is executed when the Button is clicked.

This is an example of an anonymous coderef.

You could have also used: 

    
   $mw->Button( -text => "Done", -command => &close_down )->pack;

Where close_down is a subroutine defined elsewere in the program, like: 

    
   sub close_down
    {
     exit;
    }
