4.2 Setup for a Perl/Tk script 

    
   #!/usr/bin/perl

On a Unix system, you need the #!/usr/bin/perl line to execute the script. 

    
   use Tk;

Load the Tk module. 

    
   use strict;
   use warnings;

Turn up the compiler checking to reduce stupid mistakes.
