7.6.1 Add Checkbuttons 

    
  my $f2 = $f->Frame( -label => 'Options' );

  my @optlist = qw[i s m];
  for(my $i = 0; $i < @optlist; ++$i)
   {
    $f2->Checkbutton( -text => $optlist[$i],
                      -variable => $option[$i],
                      -onvalue=>$optlist[$i],
                      -offvalue=>'' )->pack( -side => 'left' );
   }
  $f2->pack( -side => 'right', -anchor => 'n' );

I find it easier to build groups of widgets with lists and loops than to repeat code.
