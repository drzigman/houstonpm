19 Fetch as an Array 

    
  # Fetch row as an array
  @row = $sth->fetchrow_array()
    Good for processing a row as a unit
    Good for passing the whole row to a processing routine
    Only uses memory for one row at a time
    Really useful if the code has to process an unknown number of columns
    Not as obvious for processing on the spot
