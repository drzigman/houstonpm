11 What If We move to a different RDBMS?

For this simple an example, we only need to change one line if the databases are set up identically.

For MySQL, replace 

    
	my $dbh = DBI->connect( "dbi:CSV:f_dir=db", "fred", "secret" )
	    or die "Cannot connect to database: $DBI::errstrn";

with 

    
	my $dbh = DBI->connect( "DBI:mysql:db", "fred", "secret" )
	    or die "Cannot connect to database: $DBI::errstrn";

For Oracle, use 

    
	my $dbh = DBI->connect( "dbi:Oracle:dbtnsname", "fred", "secret" )
	    or die "Cannot connect to database: $DBI::errstrn";
