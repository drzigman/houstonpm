SelfLoader module

The __DATA__ token is also used to allow you to create modules that use the SelfLoader module to autoload subroutines on demand. If your module uses the SelfLoader module, then any subroutines that aren't necessary for the module to be compiled, and which aren't often used may be placed after the __DATA__ token. SelfLoader will read and parse these, but it will only compile them into perl's internal format if they're actually called in the program that uses your module. This can speed up the initial compilation phase of the script, but slightly slow down the first call to any selfloaded subroutine.

Information on the __DATA__ and __END__ tokens can be found in the perldata manpage, as well as in the documentation for the SelfLoader module.
